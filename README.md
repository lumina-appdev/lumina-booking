# lumina-booking

A full-stack booking and appointment management app built for small businesses in Australia.

Clients can book appointments online, and business owners can manage their schedule from a simple dashboard.

## Tech Stack
- **Frontend:** Next.js, Tailwind CSS
- **Backend:** Spring Boot
- **Database:** Supabase

## Features
- Online booking with time slot selection
- Business owner dashboard
- Email confirmation
- Mobile-first design
